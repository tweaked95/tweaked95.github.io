The Coloring Book

The Coloring Book is a 2D puzzle platformer in which the player uses a colour picker to swap between colours which in turn changes how the game looks and how the hazards react to the player.

The player has to complete the levels to finally acquire all the colours and colourize their world. These colors grant special enhancements and allow interaction with blocks of a particular color. The Coloring Book was developed in Unity using C# over the span of one year.

Responsibilities as a Game Designer

  • Conceptualized, documented, and executed the project lifecycle from ideation through post-production.
  • Developed core gameplay mechanics, systems, and features for a seamless player experience.
  • Added a progressive unlock system to keep players engaged.
  • Created levels with a deliberate progression curve, balancing tense and relaxed gameplay moments.
  • Optimized performance and identified bottlenecks for smooth gameplay.
  • Conducted audience research to improve game accessibility.

Role: Game Designer
Tech: Unity | C# | Solo Project
Duration: Sept 2020 – Sept 2021
